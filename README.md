## Running Dev Environement

Dev are in the ```src``` repository

# Requirement :

```
  Grunt
  Compass
  Npm
```

##To start :

1. Clone the project
2. Run ```npm install```
3. Run ```npm start```
4. Go to ```localhost:3000``` on your browser.

##To build :

Run ```npm run build```
